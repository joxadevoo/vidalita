# Public Assets

Bu papkaga statik fayllarni joylashtiring.

## ID Card Background Image

ID card orqa foni uchun rasmni quyidagi nom bilan joylashtiring:
- **Fayl nomi:** `id-card-background.jpg` yoki `id-card-background.png`
- **Joylashuv:** `frontend/public/id-card-background.jpg`

### Tavsiya etilgan o'lchamlar:
- **Kenglik:** 125mm (472px @ 96 DPI)
- **Balandlik:** 88mm (331px @ 96 DPI)
- **Format:** JPG yoki PNG
- **Ruxsat:** 300 DPI (chop etish uchun)

### Qanday qo'shish:
1. Rasmni `frontend/public/` papkasiga ko'chiring
2. Fayl nomini `id-card-background.jpg` yoki `id-card-background.png` qiling
3. Brauzerni yangilang

**Eslatma:** Agar rasm bo'lmasa, gradient fon ishlatiladi.
